ゲームの流れ 
--
プレイヤー人数は2人(赤、青)で、雪玉を投げて <br>
誰が先に相手を倒すかがこのゲームの勝利条件です。 

1. コインを利用して誰が先攻するかを決定。<br>
例）赤（攻撃） -> 青（防御） - 最初のターン<br>
   青(攻撃) -> 赤(防御) - 二番目のターン<br>

2. キーボード操作 <br>
攻撃/防御ともに同じ方向を共有し、各プレイヤーは相手の経路を予想する必要がある。 <br>
ブルー: Z,C<br>
赤:<,>(方向キー)<br>
