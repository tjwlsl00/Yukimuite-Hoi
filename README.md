タイトル：ゆきむいて、ホイ！　／　開発期間：25年12月17日 ～ 
--
赤と青は幼い頃から仲の良い友達です。 勝負が好きだった赤は雪が降ったある日青に雪合戦を申し込むようになるが.. 赤と青の息詰まる雪合戦！ 相手の攻撃、回避経路を予測し、相手より先に雪合戦で勝ってみてください！

ゲームの流れ 
--
プレイヤー人数は2人(赤、青)で、雪玉を投げて誰が先に相手を倒すかがこのゲームの勝利条件です。 

①コインを利用して誰が先攻するかを決定。<br>
例）赤（攻撃） → 青（防御） - 最初のターン<br>
青(攻撃) → 赤(防御) - 二番目のターン<br>

②キーボード操作 <br>
攻撃/防御ともに同じ方向を共有し、各プレイヤーは相手の経路を予想する必要がある。 <br>

＊方向選択(左右)<br>
赤:←、→<br>
青: Z、C<br>

＊アイテム選択<br>
解放：青(Ｓ)、赤(↑)<br>
選択：青(１番：Ｚ，２番：Ｘ，３番：Ｃ），赤（１番：←，２番：↓，３番：→）<br>

＊アイテム情報<br>
1番：雪玉のダメージを2倍増幅させます。<br>
2番：相手の雪玉に当たった場合、そのダメージを相手にそのまま返します。<br>
3番：自分と相手のターンを変えることができます。(緊急な時に使ってみてください！)<br>
