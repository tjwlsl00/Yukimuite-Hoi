タイトル：助けて、プルン！　／　開発期間：25年10月30日 ～ 25年11月25日
--
ある日と変わらず友達と冒険をしていた「プルン」。 
急にドーンという音と共に後ろを振り向くと、さっきまでそばにいた友人たちがモグラの群れに捕まって連れていかれていた。 友達は「プルン」に助けを求めるようになるが.. 果たしてプルンはモグラたちに捕まることなく無事に友達を救出できるのか？

ゲームの流れ 
--
「ぷるん」は変身能力を持っている。

①緑: 基本状態。 移動速度とジャンプ能力が最も高く、基本的な障害物突破に使用できる。 <br>
②赤: ダッシュ能力。 一定距離を突進し、マップに存在する特定のオブジェクト(箱)を破壊することができる。 <br>
③黄色: 接着能力。 マップに存在する特定の区域を遡ることができる。<br>
④青:縮小能力。 緑、赤、黄色の状態で通れない区域を通ることができる。<br>

＊移動速度: 緑 > 赤 > 黄 = 青<br>
＊ジャンプの高さ: 緑 > 赤 (黄色、青はジャンプが不可能)<br>

各能力を活用してマップの構造物を攻略し、無事に友達を救出してください！
